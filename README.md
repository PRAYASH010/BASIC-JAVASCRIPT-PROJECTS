# Basic JavaScript Projects

Welcome to the **Basic JavaScript Projects** repository! This collection features simple, beginner-friendly web development projects built using HTML, CSS, and vanilla JavaScript. These projects focus on core scripting logic, DOM manipulation, event listeners, and basic styling.

---

## 📁 Repository Structure

```text
├── AgeCalculator/
├── Analog Clock/
├── BMICalculator/
├── backgroundChanger/
├── classicCounter/
├── currency converter/
└── password generator/
```
## 🚀 Projects Included
1. Age Calculator (/AgeCalculator)
A utility application that calculates a user's exact age based on their inputted date of birth.

2. Analog Clock (/Analog Clock)
A dynamic web-based analog clock that uses CSS transitions and JavaScript's time methods to reflect the current local time in real-time.

3. BMI Calculator (/BMICalculator)
A simple health calculator that takes a user's weight and height inputs, computes their Body Mass Index (BMI), and displays their current weight status classification.

4. Background Changer (/backgroundChanger)
An interactive project that changes the background color of the webpage dynamically, utilizing a random hexadecimal color generator.

5. Classic Counter (/classicCounter)
A fundamental JavaScript application featuring buttons to increase, decrease, or reset a numerical counter value on the screen.

6. Currency Converter (/currency converter)
A practical tool designed to accept user amounts and convert monetary values from one currency denomination to another using conversion rates.

7. Password Generator (/password generator)
A security utility that automatically generates a strong, random password string based on specified configurations like character length and type.

## 🛠️ How to Run Locally
Since these are fundamental client-side web applications, you don't need any complex build configurations or servers to run them!

Clone the repository:

Bash
git clone [https://github.com/PRAYASH010/BASIC-JS-PROJECTS.git](https://github.com/PRAYASH010/BASIC-JS-PROJECTS.git)

Navigate into the specific directory of the project you want to run.

Open the index.html file inside that folder directly in any web browser (Chrome, Firefox, Edge, etc.), or use the VS Code Live Server extension for real-time changes.
